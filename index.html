Hi!  I'm SumRanDoh

I make games.


A roguelike deckbuilder on a factory floor

Your deck is factory hardware

Each run drops you onto a 7x7 factory floor with a steadily growing quota. Draft components, place them strategically, and run the machine.

The floor shifts between runs

Space is limited. Challenges disable what worked last time. You're always adapting.

Deploy your crew

Specialized worker bots amplify your production in powerful ways. Rearrange the crew anytime.

Seeds encode runs: save, replay, transmit

Every run has a seed: a compact code that recreates the same sequence of opportunities. Save it. Replay it. Transmit it. Compare results.

3/5/2025: Singularity

    I've discovered Cursor. It's a 'fork' of my favorite software for programming: VSCode, which in this case means Cursor lets me write code with an AI that has access to the folder I'm coding my game in. And it has made all the difference. Like, holy shiznit. It feels like I was debugging Prismatics as one would clean a bathroom with a toothbrush and someone just handed me a power washer.

    One of my big goals with Prismatics from the start has been to make the game's score ceiling as high as possible for the sake of Endless Mode. But the scores Snailman and I were achieving in testing were too high to be stored in single variables in the Godot file: huge exponents like 2.34*1018 that needed to be accurately compared to other huge exponents like (2.34*1018)+2. It made sense to me that solving this was possible with some fancy math that would break my feeble mind, so it hung over me as a stretch goal as I made the rest of the game.

    Since the rise of ChatGPT, the extent to which I had been able to get AI to help me finish this kind of ambitious mechanic was copying snippets of code to and from my ChatGPT window. It will write you a great file to copy-paste if you describe exactly what you need perfectly along with re-describing your entire codebase everytime. Cursor is essentially a ChatGPT that flawlessly abstracts that all away. I don't need to describe my codebase well, or even at all. I don't really need to know how it's organized anymore.

I don't even need to learn to use Cursor as much as ask it to cater to me.

"When I tell you to 'ship it', push all this to my Github with a nice change summary." Done. Macro in-place. No issues since. Seamless.

"Have this code save some logs that would help you fix it and I'll run it again." Discovering this workflow was the power washer moment. I burned through my entire backlog of bugs in a week.

"I need you to remember this process of debugging through logs that you write and read yourself."

"The Collection scene is back in scope. Can you fill it out with the unlocking behavior based on the notes I wrote back in the day?"



1/7/2026: Art and Music!

    The factory-builder deck-builder I've been working on is nearly ready for early access! All the AI-generated assets were super helpful for getting the project off the ground quickly, but I'm happy to report that I've had the chance to replace them all with actual pixel art. I've found ChatGPT and the like to be incredibly helpful for learning to use complex specialized software like Aseprite for pixel art, Audacity for sound editing, or Logic for music composition. It used to be too easy to get stuck trying to understand advanced functionality that wasn't addressed directly with great documentation or in a Stack Overflow post somewhere. But not only can modern AI models help you get unstuck in unfamiliar software, they can also recommend useful shortcuts and hidden features. When you can feel like a power-user within 15 minutes of installing new software the world of open-source software is a lot more inviting and useful.



8/19/2025: Prismatics

    I have been working on a factory-builder deck-builder, which I'm calling, "Prismatics". I had sketched up the idea in Pygame, but when it started showing promise I switched over to Godot so that I'd eventually be able to publish it beyond Steam. Right now it has no sound, and all the image assets are AI-generated placeholders, but the core gameplay is super interesting & pretty unique-feeling. I've reached out to a few publishers in case of any early bites, but I'll probably end up trying to illustrate things myself next.
