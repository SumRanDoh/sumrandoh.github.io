1/7/2026: Art and Music!

    The factory-builder deck-builder I've been working on is nearly ready for early access! All the AI-generated assets were super helpful for getting the project off the ground quickly, but I'm happy to report that I've had the chance to replace them all with actual pixel art. I've found ChatGPT and the like to be incredibly helpful for learning to use complex specialized software like Aseprite for pixel art, Audacity for sound editing, or Logic for music composition. It used to be too easy to get stuck trying to understand advanced functionality that wasn't addressed directly with great documentation or in a Stack Overflow post somewhere. But not only can modern AI models help you get unstuck in unfamiliar software, they can also recommend useful shortcuts and hidden features. When you can feel like a power-user within 15 minutes of installing new software the world of open-source software is a lot more inviting and useful.



8/19/2025: Prismatics

    I have been working on a factory-builder deck-builder, which I'm calling, "Prismatics". I had sketched up the idea in Pygame, but when it started showing promise I switched over to Godot so that I'd eventually be able to publish it beyond Steam. Right now it has no sound, and all the image assets are AI-generated placeholders, but the core gameplay is super interesting & pretty unique-feeling. I've reached out to a few publishers in case of any early bites, but I'll probably end up trying to illustrate things myself next.
